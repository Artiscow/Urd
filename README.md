# Urd

Urd er trolig den eldste av de tre nornene i norrøn mytologi som sitter ved foten av Yggdrasil og bestemmer æsenes skjebner. Sammen sitter nornene og bestemmer skjebnene til ærnene, det sies at de risser inn skjebnene i trestykke, eller spinner og kutter skjebnestråden.
Urd er verktøyet for å spinne og risse inn din egen nettside utifra treets rot.
